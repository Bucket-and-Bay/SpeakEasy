## Usage

> Clone the repo.

> Install all the dependencies.

> Run webpack -w
  
> Make sure that you have a mongo database running. Try command mongod.
  
> Start the back end server. From root directory: nodemon server/server.js.

## Requirements

- Node 4.2.x
- Mongo
